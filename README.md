# BRAINFUCK INTERPRETER
A Brainfuck interpreter written in C
## How to use it:
   1. Clone the repository
   ```sh
   git clone https://github.com/Vgwws/bf-interpreter.git
   ```
   2. Build the package
   ```sh
   mkdir -p build && cd build
   cmake ..
   make
   ```
   3. Install it
   ```sh
   sudo make install
   ```
   4. Using it
   For interpreting file:
   ```sh
   bf <file>
   ```
   For inline execution:
   ```
   bf
   ```
